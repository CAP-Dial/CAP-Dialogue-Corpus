# CAP-Dial
### A Bilingual (Chinese-English) Dataset for Psychiatric Consultations in Adolescent Mental Health

## 📌 Overview
This repository contains the metadata, documentation, and access protocols for the **CAP-Dial** dataset. The dataset consists of 350 bilingual psychiatric consultation sessions, specifically focused on adolescent mental health, with expert-level annotations for clinical topics.

Due to the sensitive nature of psychiatric consultations, the full dataset is under **restricted access** to protect patient privacy.

---

## 🔐 Data Access Instructions

To obtain access to the dataset, please follow the formal application process outlined below:

### 1. Download the Agreement
Download the **End-User License Agreement (EULA)** provided in this repository:
👉 [./Ens-User License Agreement.pdf](./Ens-User License Agreement.pdf)

### 2. Prepare Your Application
Fill out the agreement with the following information:
* Full Name and Position.
* Institutional/University Affiliation.
* A brief description of your research purpose.
* **Handwritten or Digital Signature.**

### 3. Submit Your Request
Email the signed scanned copy (PDF) to:  
📧 **751163520@qq.com** **Subject Line:** `[Data Request] Psychiatric Dataset Application - [Your Name/Institution]`

### 4. Verification and Access
Once your application is verified (usually within 5-7 business days), you will receive a **private access link** via email to download the data from Figshare.
